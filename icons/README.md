## USAGE


## Python
Use the uitls.py to get icon url for python
Example usage: <br>
```print(VaisalaIconURL.get_icon_url(10011))```  
or <br>
```print(VaisalaIconURL.get_icon_url("10011"))```  


## Javascript 
Use utils.js to get icon url for javascript codes <br>
```console.log(VaisalaIconURL.getIconURL("10010"));```<br>
or <br>
```console.log(VaisalaIconURL.getIconURL(10010));```
